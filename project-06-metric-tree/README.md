**Goal**: Create a BPMN diagram of the target business process and develop a complete metric tree. All metrics must be fully described in an Excel file to ensure transparency, consistency, and traceability.

**Role**: Process Analyst - responsible for modeling the end-to-end process in BPMN, defining KPIs, structuring the metric hierarchy, documenting formulas and data sources in Excel, and validating results with stakeholders.

**Tools**: BPMN 2.0 standard; Microsoft Excel

**Key Results**: Completed BPMN diagram of the target process; structured metric tree aligned with business goals; Excel file containing metric definitions, formulas, units, owners, data sources, and update frequency; clear linkage between process steps and measurable indicators; foundation for performance monitoring and reporting.

# Analytical & Technical Portfolio  

This repository showcases selected projects completed during my time as a Product Analyst at Quarks.  
They reflect both analytical expertise (data analysis, experimentation, modeling) and technical competencies (process design, system logic validation, structured problem-solving) relevant to roles in Data Analytics, Product Analytics, and Technical Support.

Each project folder includes an anonymized report (PDF), dashboard screenshots (where applicable), and a short project description.

---

## Contents  

- **project-01-fraud-analysis** — Audience structure & segmentation analysis  
- **project-02-ab-test** — A/B test evaluation and funnel performance comparison  
- **project-03-user-group-research** — Research on how different user groups influence each other in two-sided markets, based on analytical and scientific sources  
- **project-04-tableau-visualization** — Interactive dashboard for monitoring user behavior and key metrics  
- **project-05-BPMN-diagram** — Business process modeling using BPMN to document and validate workflow structure  
- **project-06-metric-tree** — Design of a hierarchical metric tree aligned with business goals, including full KPI documentation  
- **project-07-experiment-results-analysis** — Validation of experiment logic with event-level analysis and discrepancy identification  
- **project-08-growth-model** — Scenario-based financial growth model using retention-based projections and sensitivity analysis

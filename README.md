They reflect both **analytical skills** (data analysis, visualization, hypothesis testing) and **technical competencies** (problem-solving, system understanding, process optimization) that are relevant for roles in **Data Analytics** and **Technical Support**.

Each project folder includes an anonymized report (PDF), screenshots of dashboards, and a short description.

**Goal**: Design and document a clear, standardized authorization workflow using BPMN to ensure consistent access control, reduce security risks, and improve process transparency.

**Role**: Process Analyst - gather business and security requirements; model the authorization flow in BPMN; identify decision points, roles, and system interactions; validate the process with stakeholders

**Tools**: BPMN 2.0 standard

**Key Results**: Clearly defined authorization workflow with roles and responsibilities; Identified approval levels and decision gateways
